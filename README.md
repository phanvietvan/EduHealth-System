# EduHealth System 🏥

Hệ thống quản lý y tế học đường hiện đại, tích hợp AI và trải nghiệm người dùng cao cấp.

## Công nghệ sử dụng
- **Frontend**: HTML5, Tailwind CSS (via CDN), Google Fonts (Manrope).
- **Design System**: Glassmorphism, Cyber-glow effects.
- **Icons**: Material Symbols Outlined.

## Cấu trúc thư mục hiện tại
- `index.html`: File chính chứa cấu trúc trang web.
- `src/style.css`: File chứa định nghĩa các style tùy chỉnh và animations.

## Bắt đầu phát triển
1. Mở `index.html` trực tiếp trong trình duyệt hoặc sử dụng Live Server.
2. Tùy chỉnh các thành phần (Components) trong `index.html`.
3. Chỉnh sửa style tại `src/style.css`.
